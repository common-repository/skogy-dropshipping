Sell products on your woocommerce store without investing in inventory

== Description ==
This plugin connects and opens an account for your woocomerce store for skogy dropshipping. 

Browse and select products you are interested in selling, click "add to products". Simple as that.

[youtube https://www.youtube.com/watch?v=PPMaD7fnO8A]

= Features =

* No setup / Deposit
* Trusted suppliers
* Massive catalog of products
* Place dropshipping orders in bulk

== Installation ==

= Automatic Installation =
Go to WordPress dashboard, click  Plugins / Add New  , type 'skogy dropshipping' and hit Enter.
Install and activate plugin, visit WooCommerce > Skogy dropshipping.

= Manual Installation =
[Please, visit the link and follow the instructions](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)
